# HAT-P-36 Analysis

This project presents a full analysis of the HAT-P-36 system, including:

- Spectroscopic reduction of the host star (HAT-P-36)
- Transit photometry analysis of HAT-P-36 b
- Estimation of stellar temperature and planetary radius

## Structure

- `data/` — raw observational data (FAST + KeplerCam)
- `notebooks/` — analysis pipelines
- `results/` — plots and outputs

## Notes

This repository is designed to be fully reproducible. All file paths are relative and the notebook can be run top-to-bottom.
